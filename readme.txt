ESTADO DEL SISTEMA: VINCULADO / ENCRIPTADO [AES-256]
NIVEL DE ACCESO: GÉNESIS (ORIGIN LEVEL)

Este software es una terminal de escritorio de alta precisión diseñada para 
el análisis de vectores estratégicos y sincronización de timing ejecutivo.

---------------------------------------------------------------------------
INSTRUCCIONES DE INSTALACIÓN Y ACTIVACIÓN:
---------------------------------------------------------------------------

1. DESCOMPRESIÓN:
   Extraiga el contenido de este archivo .ZIP en una carpeta local de su 
   unidad de disco duro (se recomienda C:\VaultLogic o Escritorio).

2. EJECUCIÓN:
   Localice el archivo "Vault-Logic.exe" (o el acceso directo generado) 
   y ejecútelo con permisos de usuario estándar.

3. VÍNCULO DE IDENTIDAD:
   Al iniciar por primera vez, la terminal solicitará su "Email de Registro".
   Ingrese la dirección de correo con la que adquirió su licencia GÉNESIS. 
   El sistema sincronizará sus vectores natales con la base de datos de 
   precisión de la NASA JPL de forma automática.

---------------------------------------------------------------------------
NOTAS TÉCNICAS:
---------------------------------------------------------------------------
* CONEXIÓN: Se requiere acceso a internet para la validación de integridad 
  de los efemérides planetarios en tiempo real.
* PRIVACIDAD: Sus datos de nacimiento y coordenadas están protegidos por 
  protocolos de encriptación de grado bancario.
* SOPORTE: En caso de desincronización de la terminal, reinicie la aplicación 
  o contacte a su analista asignado.
